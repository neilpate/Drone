# Hardware/

Physical design files for the drone, split by engineering discipline.

- [`Mechanical/`](Mechanical/) — Fusion 360 sources, STEP exports, print-ready meshes ([ADR 0006](../Doc/decisions/0006-mechanical-cad-fusion360.md)).
- [`Electrical/`](Electrical/) — KiCad schematics, board layouts, fabrication outputs for the custom nRF5340 PCBA (Phase 4 onwards).

Each subfolder has its own README with file-format and naming conventions.
